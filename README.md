![tetete](http://goput.it/lz5.png)

Make a symlink called tewithink to tewisay to use the thinking balloon, or
use the -b option.

Features
--------
* Ignores ANSI escape colors when getting width
* Finds correct width on all unicode characters
* More balloon styles
* Tewis
